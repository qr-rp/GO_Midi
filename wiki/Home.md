# GO_MIDI! 使用文档

欢迎来到 GO_MIDI! 使用文档。GO_MIDI! 是一款强大的 MIDI 文件播放器，可以将 MIDI 音乐转换为键盘按键，实现在游戏中自动演奏。

## 文档目录

### 基础使用
- [快速开始](快速开始) - 从安装到第一次播放
- [播放列表管理](播放列表管理) - 导入、管理和切换 MIDI 文件
- [通道配置](通道配置) - 设置目标窗口、音轨和移调

### 键位映射
- [键位映射](键位映射) - 自定义音符到按键的映射关系

### 高级功能
- [定时播放](定时播放) - NTP 时间同步与定时开始
- [高级功能](高级功能) - 变速、和弦分解、延迟补偿等

### 其他
- [常见问题](常见问题) - 使用中遇到的问题及解决方案

## 主要特性一览

| 功能 | 描述 |
|------|------|
| 多音轨支持 | 完整解析 MIDI 文件中的所有音轨 |
| 智能移调 | 自动计算最佳移调，让音符落在可用键位范围内 |
| 8 通道配置 | 每个通道可独立配置目标窗口、音轨和移调 |
| 多播放模式 | 单曲、列表、随机、循环等多种播放模式 |
| 全局热键 | F12 全局播放/暂停 |
| 定时播放 | NTP 精确时间同步，支持定时开始 |
| 键位自定义 | 支持音名和 MIDI 编号两种映射格式 |
